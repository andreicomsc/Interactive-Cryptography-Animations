Animation Input Data

The below input fields should:

  1. Contain bits of "0" and "1" only.
  2. Be of the same length, and contain at least 10 bits each.
  3. The Encoding and Measuring bases should have index wise, at least 4 matching bits.

Initial Key:

0 - denotes a classical binary "0" Key bit, transmitted from Sender to Receiver.

1 - denotes a classical binary "1" Key bit, transmitted from Sender to Receiver.

Encoding Basis:

0 - denotes that the Initial Key bit is Encoded by the Sender using a Rectilinear Basis.

1 - denotes that the Initial Key bit is Encoded by the Sender using a Diagonal Basis.

Measuring Basis:

0 - denotes that the received Key bit is Measured by the Receiver using a Rectilinear Basis.

1 - denotes that the received Key bit is Measured by the Receiver using a Diagonal Basis.

Eavesdropper status:

0 - denotes that the Eavesdropper is Passive during this Key bit transmission.

1 - denotes that the Eavesdropper is Active during this Key bit transmission.
